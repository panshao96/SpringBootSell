# SpringBootSell
一个外卖平台

### 环境
* JDK 1.8.0_111
* Maven 3.3.9
* SpringBoot 2.2.1
* mySQL 8.0.18
* redis

### 下载
```
git clone https://github.com/panshao96/SpringBootSell.git
```

### 数据库建表
* 查阅根目录下SQL.md文件

### 打包
```
mvn clean package -Dmaven.test.skip=true
```

### 部署
```
java -jar 'jar包的绝对路径'
```
